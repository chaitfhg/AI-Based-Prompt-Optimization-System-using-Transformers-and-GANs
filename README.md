🚀 Transformer + GAN Prompt Optimizer

An advanced AI-powered prompt optimization system that combines Transformer Neural Networks and Generative Adversarial Networks (GANs) to automatically generate, evaluate, and improve prompts for multiple domains and use cases. The application features an interactive Streamlit dashboard with real-time prompt scoring, GAN training visualization, prompt evolution tracking, and export functionality.

✨ Features
Transformer-based Prompt Generator and Prompt Discriminator
GAN-driven prompt mutation and optimization pipeline
Multi-criteria prompt evaluation and scoring system
Domain, tone, and output format customization
Interactive Streamlit dashboard with analytics and charts
Real-time GAN training metrics and score evolution tracking
Export optimized prompts in CSV and JSON formats
Fully offline implementation using PyTorch and Streamlit
